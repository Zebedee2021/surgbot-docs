    CR5 控制接口¶

    越疆 CR5AF 机械臂¶

    越疆 CR5AF 是 CR5 系列的力控版本，是一款 5kg 负载的力控型协作机械臂，内置力/力矩传感器，支持柔顺拖动示教和恒力控制，适合医疗辅助场景的精细操作。

    参数 规格
    型号 CR5AF
    负载 5kg
    自由度 6 轴
    重复定位精度 ±0.02mm
    力控特性 柔顺拖拽、恒力控制、碰撞检测
    工作环境 协作场景，无需安全围栏

    AF 后缀含义

    AF = Advanced Force，表示该型号内置六维力/力矩传感器，原生支持力控模式，相比标准 CR5 增加了力觉感知能力。

    控制方式¶

    • 通过 Python 直接调用越疆 Dobot SDK
    • TCP 通信连接机械臂控制器
    • 支持关节空间控制和笛卡尔空间控制

    SDK 与通信接口¶

    项目 说明
    SDK 语言 Python / C++ / C#
    通信协议 TCP/IP
    默认端口 8080
    ROS 支持 ROS1 / ROS2 生态

    当前项目采用 Python SDK + TCP 直连 方案。

    参数配置需求¶

    根据现场测试反馈，以下参数需要支持通过 GUI 界面配置：

    参数 说明
    机械臂速度 运动速度百分比
    Z 轴偏移量 高度补偿值
    夹持力大小 夹爪夹持力
    力反馈大小 力反馈阈值

    配置文件

    所有参数应保存到 YAML 配置文件中，支持通过界面修改后实时保存，避免每次修改代码。

    CR5AF 力控优化¶

    器械夹取力控¶

    通过 set_force_threshold 接口设定夹持力阈值，让机械臂在夹取器械时自动限制力度，避免夹坏精密器械或碰伤人员。

    低速精准递交¶

    医疗场景对运动平稳性要求高，建议将运动速度设定为较低值（如 speed=0.2），确保器械递送过程平稳可控。

    碰撞信号监测¶

    CR5AF 内置碰撞检测功能，可通过 SDK 监听 collision 信号。一旦触发碰撞检测，机械臂立即停止运动并自动回到预设安全姿态。详见 安全策略。

    手术场景特殊要求

    不同器械的材质和重量差异大（轻如缝针、重如骨钳），力阈值需按器械类型动态调整，不能使用统一的固定值。建议在配置文件中为每类器械预设对应的力控参数。
