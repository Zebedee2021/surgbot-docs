# 领域知识

## 手术器械护士职责

手术器械护士（又称刷手护士、Scrub Nurse）是手术室护理团队中的核心角色，负责手术器械的管理和传递。

### 工作流程

#### 术前准备

- 术前复习患者病情、手术步骤，参与术前讨论
- 提前 15 分钟洗手并准备器械台
- 分类摆放器械，检查导管等耗材
- 建立并维护无菌区域

#### 术中执行

- 精准传递器械给主刀，保持把手朝外、姿势一致、防止污染
- 密切观察手术进程，预判医生下一步所需
- 配合巡回护士进行器械清点（切开体腔前、关闭前后双重清点）
- 管理手术台上的器械，确保摆放有序

#### 术后整理

- 协助清点所有器械和纱布，确保数量无误
- 将使用过的器械分类、初步清洗
- 清理手术台，协助处理医疗废物
- 填写器械清点记录

### 器械台布局规范

器械按功能分区摆放：

| 区域 | 器械类型 |
|------|---------|
| 持物区 | 持针钳、止血钳 |
| 切割区 | 手术刀柄、剪刀 |
| 缝合区 | 缝线、持针器 |
| 其他 | 特殊器械单独放置 |

### 递交器械规范

- 传递器械时将器械尖端朝向自己，手柄递给医生
- 清晰报出器械名称
- 与巡回护士协同，在手术开始、缝合前和结束前进行三次清点

## 手术室站位

不同类型手术有不同的人员站位和设备布局要求。主要科室包括：胸外科、普外科、耳鼻喉科、血管外科、泌尿外科、妇产科、眼科、神经外科、脊柱外科、关节外科等。

关键要素：

- 手术床位置与角度
- 主刀医生站位
- 器械护士站位
- 巡回护士活动区域
- 设备（腔镜台车、显微镜等）摆放位置
- 器械台与手术床的相对位置

## 人机协同设计参考

机器人器械护士的引入应遵循以下原则：

1. **机器人负责**：高频、可预测、重复性动作（识别器械、稳定抓取、精确递交、计数留痕）
2. **护士负责**：术前台面规划、术式判断、异常决策、监督与最终控制
3. **协同点**：递交的机械行为统一规范（把手姿态、递交流程、急停规范），由护士触发与把关，机器人执行并自动记录
