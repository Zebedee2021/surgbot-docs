# 硬件平台

## 服务器

| 配置项 | 规格 |
|-------|------|
| GPU 1 | NVIDIA RTX 4090 (48GB 显存) |
| GPU 2 | NVIDIA Titan X (24GB 显存) |
| 系统 | Ubuntu Linux |
| 用途 | 模型训练、推理、仿真 |

!!! note "当前限制"
    两张显卡当前无法联合使用，项目采用单卡运行方案，主要依赖 RTX 4090 完成模型加载与推理。

## 机械臂

| 配置项 | 规格 |
|-------|------|
| 型号 | 越疆（Dobot）CR5 系列协作机械臂 |
| 自由度 | 6 轴 |
| 末端执行器 | 双指夹爪（力/位传感） |
| 控制方式 | Python SDK (TCP) |
| 安全特性 | 碰撞检测、力反馈、急停 |

## 视觉传感器

| 设备 | 数量 | 用途 |
|------|------|------|
| 俯视 RGB-D 摄像头 | 2 | 器械台俯拍，器械检测与定位 |
| 腕部相机（可选） | 1 | eye-in-hand 近距离精确识别 |

## 手术室环境

根据现场照片，雄安宣武医院手术室（神经外科）配备：

- 蔡司手术显微镜
- 多屏影像显示系统（CT/MRI 显示）
- 导航系统
- 标准手术床（可调节高度/角度）
- 无菌器械台（蓝色无菌布覆盖）

!!! warning "空间约束"
    手术室设备密集（显微镜、监护仪、导航系统、C 臂等），机械臂工位和摄像头安装位置选择受限，需要在不影响医护人员操作空间的前提下布置。
