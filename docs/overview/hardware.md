# 硬件平台

## 服务器

| 配置项 | 规格 |
|-------|------|
| GPU 1 | NVIDIA RTX 4090 (48GB 显存) |
| GPU 2 | NVIDIA Titan X (24GB 显存) |
| 系统 | Ubuntu Linux |
| 用途 | 模型训练、推理、仿真 |

!!! note "当前限制"
    两张显卡当前无法联合使用，项目采用单卡运行方案，主要依赖 RTX 4090 完成模型加载与推理。

## 机械臂

| 配置项 | 规格 |
|-------|------|
| 型号 | 越疆 CR5AF（CR5 系列力控版本）协作机械臂 |
| 负载 | 5kg |
| 自由度 | 6 轴 |
| 重复定位精度 | ±0.02mm |
| 末端执行器 | 双指夹爪（力/位传感） |
| 控制方式 | Python/C++/C# SDK (TCP/IP, 端口 8080) |
| 力控特性 | 柔顺拖动示教、恒力控制、碰撞检测 |
| 安全特性 | 碰撞检测、力反馈、急停 |

### 机械臂开发接口

| 项目 | 说明 |
|------|------|
| SDK 语言 | Python / C++ / C# |
| 通信协议 | TCP/IP |
| 默认端口 | 8080 |
| ROS 支持 | ROS1 / ROS2 |
| 推荐操作系统 | Ubuntu 20.04 / 22.04 |

!!! note "项目选型"
    当前项目采用 **Python SDK + TCP 直连** 方案，运行于 Ubuntu Linux 服务器，通过 VS Code Remote-SSH 进行远程开发。详见 [开发环境搭建](../guides/env_setup.md)。

## 手术室环境

根据现场照片，雄安宣武医院手术室（神经外科）配备：

- 蔡司手术显微镜
- 多屏影像显示系统（CT/MRI 显示）
- 导航系统
- 标准手术床（可调节高度/角度）
- 无菌器械台（蓝色无菌布覆盖）

!!! warning "空间约束"
    手术室设备密集（显微镜、监护仪、导航系统、C 臂等），机械臂工位和摄像头安装位置选择受限，需要在不影响医护人员操作空间的前提下布置。
