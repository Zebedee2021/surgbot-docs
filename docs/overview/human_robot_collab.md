# 人机协同方案

## 核心理念

机器人器械护士的定位**不是替代真人器械护士**，而是在固定工位执行部分标准化操作，与真人护士形成分工协作。

## 角色定位

### 机器人器械护士

- **工位**：固定在无菌台一侧，覆盖"器械抓取区 + 中立交接区"
- **职能**：
    - 按指令抓取并递交常用器械（Top-10 高频使用）
    - 自动记录器械取出/归还信息
    - 执行高频、重复、低决策的动作

### 真人器械护士

- **工位**：位于手术床一侧，近距离配合主刀
- **职能**：
    - 负责器械台布置、特殊/临时器械递交、清点与回收
    - 突发情况（手术方式改变、出血、器械污染）时立即接管
    - 作为"人机交互接口"，决定机器人何时介入/退出

## 配合模式

### 模式一：触发-执行模式（主导）

真人护士通过**脚踏开关/语音指令/触摸屏**发出"递交镊子"指令，机器人抓取并递交到中立区递交点，由主刀直接取用。

### 模式二：序列预测-确认模式（增强型）

机器人基于手术阶段预测下一个常用器械（例如缝合阶段 → 持针器 + 缝针），在屏幕/语音提示"下一件：持针器" → 护士确认/否决 → 机器人递交。

### 模式三：异常接管模式（安全）

若机器人动作超时、抓取失败、递交点被阻挡 → 自动回退到安全位，真人护士即时接管递交，避免手术节奏中断。

## 递交动作规范

| 项目 | 规范 |
|------|------|
| 器械方向 | 把手朝主刀、刀尖远离 |
| 递交角度 | 固定 30-45 度 |
| 停留等待 | 在 Hand-off Zone 停留，直到确认取走 |
| 释放确认 | 双确认释放（脚踏触发 + 拉力/握力传感器检测） |

## 安全机制

- **冗余设计**：机器人仅处理 70-80% 高频器械，复杂/临时器械仍由护士人工递交
- **安全边界**：机器人工位与主刀/巡回护士区域之间设定安全边界，避免跨区动作
- **应急切换**：任何时间护士可按急停或语音指令"暂停机器人"，立即转为人工递交模式

## 协同价值

- **减负**：机器人处理高频重复动作，护士减少弯腰、伸手动作，降低体力与注意力负担
- **提效**：递交流程标准化、动作一致，减少误差
- **沉淀经验**：机器人记录器械使用序列，形成数据资产，反哺培训与流程优化
