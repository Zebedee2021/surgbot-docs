## 使用指南

!!! question "如何使用本追踪表？"

    **添加新问题：** 在对应分类表中新增一行，分配 ISS 编号（递增），填写完整信息。

    **更新状态：** 直接修改状态列的 emoji 标记，同步更新统计概览中的数字。

    **标记已解决：** 将状态改为 ✅，并将该行移动到「已解决问题归档」表中，记录解决方案和日期。
