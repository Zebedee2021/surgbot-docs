# 测试记录

## 测试计划

### 测试目标

验证手术器械护士机器助手在真实手术室环境中的功能可用性、安全性和人机协同效果。

### 评测指标

| 指标类别 | 指标 | 目标值 |
|---------|------|--------|
| 功能性 | 抓取成功率 | >= 97% |
| 功能性 | 递交成功率 | >= 98% |
| 功能性 | 平均递交时延 | <= 2.5s |
| 安全性 | 急停响应 | <= 80ms |
| 安全性 | 碰撞/误释放率 | 0 |
| 用户体验 | 护士负荷降低 | >= 30% |
