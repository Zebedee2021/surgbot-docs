# 数据流水线

## 数据采集与格式

### 原始数据采集

编写数据采集代码，控制机械臂完成目标任务，同步保存运动数据和视觉数据：

- **运动数据**：关节角、末端坐标、夹爪开合度
- **视觉数据**：第三人称 RGB、末端第一人称、深度图
- **格式**：每次运动生成一个 `.npy` 文件

### 数据格式转换

将分散的 `.npy` 文件按 RLDS 格式整合：

1. 按任务整理为单文件
2. 修改 tfds 数据集构建文件（调整图片尺寸、对齐命名规范）
3. 生成训练/验证集

### 数据校验与归一化

对动作数据做归一化/标准化，匹配模型输出尺度。

## VLA 微调数据集制作

1. **注册数据集**：修改 OpenVLA 的 3 个核心文件，完成自定义数据集的注册与读取
2. **微调脚本**：配置模型路径、数据集目录、LoRA 低秩训练参数
3. **微调执行**：用 torchrun 启动，约 6000 步可达到基础效果
