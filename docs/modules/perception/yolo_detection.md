# YOLO 器械检测

## 功能说明

使用 YOLO 模型对手术器械托盘进行实时目标检测，识别器械的种类和位置。

## 当前问题

根据 2026 年 2 月现场测试反馈：

!!! warning "已知问题"
    1. **误识别**：YOLO 会将托盘以外的物体误识别为器械
    2. **同类区分困难**：不同型号的同类器械（如不同型号的骨钳）外观差异小，难以精确区分
    3. **识别精度不足**：器械朝向、夹取点识别不准，经常偏移

## 改进方向

- 解决误识别问题，限定检测区域在托盘范围内
- 扩充训练数据集，增加同类不同型号器械的样本
- 结合图像识别框、夹取点和朝向点，通过算法修正结果

## 备选方案

可以考虑在器械上打印二维码，通过扫描二维码确认器械的类型、名称、型号信息，作为视觉识别的补充。
