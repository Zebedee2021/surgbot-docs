# 相机标定

## 功能说明

相机标定包含相机内参标定和手眼标定，目的是将图像中识别到的器械像素坐标转换为机械臂工作空间中的三维坐标。

## 标定内容

| 标定类型 | 方法 | 说明 |
|---------|------|------|
| 内参标定 | 棋盘格标定法 | 获取相机焦距、畸变参数 |
| 外参标定 | 手眼标定 | 相机与机械臂坐标系映射 |
| 时间同步 | 传感器同步 | 多传感器数据对齐 |

## 当前问题

!!! warning "现场测试反馈"
    标定过程太繁琐，两个摄像头标定一次至少需要**一个小时**。需要优化标定流程，尽量做到半自动化，通过按钮点击自动记录和保存标定数据。

## 改进计划

1. 开发半自动标定工具，通过 GUI 按钮触发自动记录和保存标定数据
2. 支持标定结果保存到配置文件，避免每次重复标定
3. 实现标定结果的自动验证
