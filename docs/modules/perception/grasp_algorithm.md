# 夹取点算法

## 功能说明

根据 YOLO 检测结果，计算器械的最佳夹取位置和夹取方向，指导机械臂完成稳定抓取。

## 算法输入

- YOLO 检测框（器械边界框）
- 夹取点（白色标记点）
- 朝向点（红色和黄色连线方向）

## 当前问题

!!! warning "不可直接使用原始识别结果"
    器械朝向、夹取点识别不准，经常偏移导致夹取不到。需要结合图像识别框、夹取点和朝向点，通过算法修正夹取位置和夹取方向。

## 改进方向

需要开发修正算法，综合考虑：

1. 识别框的几何中心与夹取点的偏差修正
2. 朝向角度的平滑滤波
3. 基于器械类型的夹取策略（不同器械的最佳抓取位置不同）
4. Z 轴高度补偿（托盘不同位置高度不一致的问题）
