# 虚拟手术室方案

## 方案概述

基于 Isaac Sim + Omniverse 构建虚拟手术室，还原真实手术室环境布局。

## 架构分层

### 1. 物理建模层（Omniverse）

**环境建模**：

- 手术室空间（墙体、灯光、吊塔、监护设备）
- 手术床（可调节高度/角度）
- 无菌台 & 中立区托盘
- 医生、护士虚拟人模型

**器械建模**：

- Top-10 常用手术器械（镊子、剪刀、持针器、刀片等）
- 按真实 CAD/3D 扫描导入，赋予物理属性（质量、摩擦、碰撞体）

### 2. 动力学与感知仿真（Isaac Sim）

- 仿真刚体动力学（抓取时的摩擦、惯量、滑动）
- 限速/限力设置，验证安全性

### 3. 控制与交互

- ROS2 Bridge 接入 MoveIt! 轨迹规划
- Python API (Isaac SDK) 执行状态机
- 虚拟脚踏板、护士 GUI、语音指令（Nemo 语音识别）

### 4. 协同与沉淀（Omniverse Nucleus）

- 多用户协同接入同一场景
- 自动记录操作数据
- 生成"手术器械递交序列模型"

## 评测指标

| 指标类别 | 指标 | 目标值 |
|---------|------|--------|
| 功能性 | 抓取成功率 | >= 97% |
| 功能性 | 递交成功率 | >= 98% |
| 功能性 | 平均递交时延 | <= 2.5s |
| 安全性 | 急停响应 | <= 80ms |
| 安全性 | 碰撞/误释放率 | 0 |
| 培训效果 | 护士负荷降低 | >= 30%（NASA-TLX 评分） |
