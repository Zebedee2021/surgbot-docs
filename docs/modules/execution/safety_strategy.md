# 安全策略

## 安全需求

手术室场景对安全性要求极高，机械臂的任何异常动作都可能造成严重后果。

## 安全机制设计

### 急停机制

- 物理急停按钮
- 语音指令"暂停机器人"
- 软件层面的紧急停止接口

### 碰撞检测

- 基于力传感器的碰撞检测
- 运动前的路径安全检查
- 工作空间边界限制

### 力反馈

- 递交时通过力反馈确认医生已握住器械
- 夹取时检测是否稳定夹持

### 异常处理

!!! danger "必须解决"
    目前存在目标位置识别错误时机械臂仍然运动的安全隐患。需要在执行前综合验证目标位置的合理性，路径规划时加入安全检查。

## 安全等级

| 场景 | 响应 | 要求 |
|------|------|------|
| 急停触发 | 立即停止所有运动 | 响应时间 <= 80ms |
| 碰撞检测 | 停止并回退到安全位 | - |
| 识别异常 | 暂停执行，等待确认 | - |
| 通信中断 | 自动进入安全模式 | - |
