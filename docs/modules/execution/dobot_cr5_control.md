# CR5 控制接口

## 越疆 CR5 机械臂

越疆 CR5 是一款 6 轴协作机械臂，通过 Python SDK 进行控制。

## 控制方式

- 通过 Python 直接调用越疆 Dobot SDK
- TCP 通信连接机械臂控制器
- 支持关节空间控制和笛卡尔空间控制

## 参数配置需求

根据现场测试反馈，以下参数需要支持通过 GUI 界面配置：

| 参数 | 说明 |
|------|------|
| 机械臂速度 | 运动速度百分比 |
| Z 轴偏移量 | 高度补偿值 |
| 夹持力大小 | 夹爪夹持力 |
| 力反馈大小 | 力反馈阈值 |

!!! info "配置文件"
    所有参数应保存到 YAML 配置文件中，支持通过界面修改后实时保存，避免每次修改代码。
