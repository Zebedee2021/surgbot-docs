# DPO 对齐优化

## 背景

SFT 阶段训练后，模型仍存在以下问题：

- **误识别**：将"血压计"误识别为"血管钳"
- **冗余输出**：输出无关文本或不存在的器械名称
- **幻觉问题**：编造不存在的器械型号

## 偏好数据集构建

收集模型 SFT 阶段的错误案例，每个样本构成三元组：

- 原始文本（输入）
- 模型错误输出（rejected）
- 正确标准输出（chosen）

## DPO 训练配置

| 参数 | 值 |
|------|-----|
| 框架 | LlamaFactory DPO 模块 |
| 训练轮数 | 2 轮 |
| 学习率 | 2e-5 |

## 效果

通过 DPO 优化，使模型更倾向于输出准确的标准化器械名称，减少幻觉和冗余输出。
