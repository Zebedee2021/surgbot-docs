# 谭文韬 - 仿真工作

## 负责模块

仿真模块 — VLA 部署与 Isaac Sim 仿真

## 工作内容

参与 VLA 具身智能平台搭建，负责将 VLA 模型部署到机械臂及 Isaac 仿真环境中，并完成相关测试验证。

### 主要工作

1. **VLA 模型部署**：针对开源 VLA 模型开展复现与部署工作，尝试将模型应用到真实机械臂平台，并进行功能测试与效果验证

2. **Isaac Sim 仿真环境搭建**：负责 Isaac 仿真环境的搭建与机械臂部署，在仿真环境中完成数据采集，为后续模型训练提供支撑

3. **Sim2Real 推进**：推进 sim2real 相关验证与落地

4. **服务器环境**：实验室服务器的 VLA 镜像安装
