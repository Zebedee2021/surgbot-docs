# 陈端端 - NLP 工作

## 负责模块

NLP 模块 — 医疗器械实体识别大模型

## 工作内容

参与雄安宣武医院项目开发，核心负责院内医疗器械实体识别专用大模型的全流程训练与落地优化。模型核心目标为精准识别医疗文本中的医疗器械名称并完成标准化输出。

### 主要工作

1. **训练数据集体系化建设**：全链路主导训练数据集建设，负责医疗文本的脱敏清洗、实体标注与全流程质控，制定医疗器械实体标注规范，搭建常用器械标准化词库

2. **大模型 SFT 监督微调全流程落地**：设计全场景指令微调模板，开展多批次多轮次微调实验与超参数调优，迭代优化模型识别精度与场景适配能力

3. **DPO 直接偏好优化对齐工作**：搭建专属偏好数据集，解决模型误识别、冗余输出、幻觉等核心问题，保障模型符合业务落地标准

4. **配套工作**：同步配合完成语音指令异常排查、文字处理模块开发及安全策略规划
