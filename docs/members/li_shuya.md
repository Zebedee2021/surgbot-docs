# 李淑雅 - 感知工作

## 负责模块

感知模块 — 图像识别与夹取算法

## 工作内容

主要负责图像识别方面的内容，包括数据集和识别算法。

### 主要工作

1. **YOLO 模型优化**：解决 YOLO 模型误识别问题（如将托盘外物体识别为器械）

2. **夹取点修正算法**：开发算法，结合图像识别框、夹取点（白点）和朝向点（红黄线），计算并输出修正后的夹取位置和方向，不可直接使用原始识别结果

3. **数据集建设**：收集和标注器械图像数据集
