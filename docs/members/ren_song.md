# 任松 - 决策/感知工作

## 负责模块

决策模块 + 感知模块全链路设计

## 工作内容

### 模块划分设计

设计了三大模块的整体架构：感知模块、决策模块、执行模块。

### 第一个模块：感知模块

1. **硬件搭建与标定**：多相机部署（第三人称主相机 + 末端 wrist 相机）、相机内参/外参标定（棋盘格）、机械臂-相机手眼标定、传感器时间同步

2. **数据采集与预处理**：多传感器数据同步采集（RGB/深度图像、关节角、末端位姿、夹爪状态）、图像预处理（畸变校正、去噪、裁剪、分辨率统一）、数据格式标准化（统一为 .npy）

### 第二个模块：决策模块

1. **VLA 数据集制作**：
    - 采集 .npy 原始实时数据
    - 按 RLDS 格式整合数据
    - 数据校验与归一化

2. **OpenVLA 微调**：
    - 修改 OpenVLA 的 3 个核心文件注册自定义数据集
    - 编写微调脚本（finetune.sh），LoRA 低秩训练
    - 约 6000 步可达到基础效果

### 第三个模块：执行模块

- 根据 VLA 模型输出，通过 Python 调用越疆 Dobot CR5 SDK 实现运动控制
